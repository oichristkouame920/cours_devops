# Exercice 4 — compterMots(String phrase)

## Objectif
Écrire les tests JUnit pour la méthode `compterMots(String phrase)`
qui retourne le nombre de mots dans une phrase.

## Cas à tester
| Entrée                  | Résultat attendu |
|-------------------------|-----------------|
| "Bonjour tout le monde" | 4               |
| "Bonjour"               | 1               |
| ""                      | 0               |
| null                    | 0               |
| "un   deux   trois"     | 3               |
| "  bonjour monde  "     | 2               |

## Consigne
Créer `src/test/java/TexteUtilsTest.java` et tester tous les cas ci-dessus.
