# Exercice 2 — calculerNote(double moyenne)

## Objectif
Écrire les tests JUnit pour la méthode `calculerNote(double moyenne)`
qui retourne une mention selon la moyenne.

## Barème
| Moyenne       | Mention      |
|---------------|--------------|
| >= 16         | Excellent    |
| >= 14         | Bien         |
| >= 12         | Assez Bien   |
| >= 10         | Passable     |
| < 10          | Insuffisant  |

## Consigne
1. Ouvrir le projet `versionA/` dans IntelliJ
2. Créer `src/test/java/NoteScolaireTest.java`
3. Tester chaque mention ET chaque valeur limite

## Question
Tester versionB et versionC. Quel test détecte le bug dans chaque version ?
