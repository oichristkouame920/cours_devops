# Exercice 1 — estMajeur(int age)

## Objectif
Écrire les tests JUnit pour la méthode `estMajeur(int age)`
qui retourne `true` si age >= 18, `false` sinon.

## Consigne
1. Ouvrir le projet `versionA/` dans IntelliJ
2. Créer `src/test/java/VerificationTest.java`
3. Écrire les tests pour couvrir :
   - Un âge clairement majeur (ex: 25)
   - La valeur limite exacte (18)
   - Un âge clairement mineur (ex: 17)
   - Un âge de 0
   - Un âge négatif

## Question
Tester ensuite versionB et versionC avec les mêmes tests.
Quelle version passe tous les tests ? Pourquoi ?
